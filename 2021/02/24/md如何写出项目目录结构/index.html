md如何写出项目目录结构

文章借鉴：markdown如何写出项目目录结构和.md文件自动生成项目目录结构，预览.md文件

  • treer -e

  1. 安装treer:npm install treer -g，查看版本确定安装成功:treer --version

  2. 进入项目文件，启动cmd后输入treer -e ./result.txt.

  3. 将内容复制到md文档的代码块中，不需要说明语言。（否则会很难看）

  • mddir

  1. 安装mddir：npm install mddir -g

  2. 进入项目文件,启动cmd后输入mddir,就会生成一个directoryList.md

  3. 将内容复制到md文档的代码块中，不需要说明语言。（否则会很难看）
