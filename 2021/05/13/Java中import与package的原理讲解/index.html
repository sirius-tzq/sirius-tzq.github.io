JavaдёӯimportдёҺpackageзҡ„еҺҹзҗҶи®Іи§Ј

еңЁиҜҙpackageгҖҒimportжңәеҲ¶еүҚжҲ‘д»¬е…ҲжқҘдәҶи§ЈдёӢjavaзҡ„CLASSPATHгҖӮ

CLASSPATHйЎҫеҗҚжҖқд№үе°ұжҳҜclassзҡ„и·Ҝеҫ„пјҢеҪ“жҲ‘д»¬еңЁзі»з»ҹдёӯиҝҗиЎҢжҹҗдёӘjavaзЁӢеәҸж—¶пјҢе®ғе°ұдјҡе‘ҠиҜүзі»з»ҹеңЁиҝҷдәӣең°ж–№еҜ»жүҫиҝҷдёӘclassж–Үд»¶

CLASSPATH=.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar; иҝҷжҳҜжҲ‘д»¬javaй»ҳи®ӨзҺҜеўғеҸҳйҮҸи®ҫзҪ®гҖӮ

вҖң.вҖқзҡ„ж„ҸжҖқжҳҜеңЁеҪ“еүҚиҝҷдёӘж–Үд»¶еӨ№жүҫпјӣ%JAVA_HOME%\libжҳҜеңЁjavaе®үиЈ…и·Ҝеҫ„зҡ„libж–Үд»¶еӨ№жүҫпјӣ%JAVA_HOME%\lib\tools.jarжҳҜеңЁjavaе®үиЈ…и·Ҝеҫ„зҡ„libж–Үд»¶еӨ№зҡ„toolsеҺӢзј©еҢ…дёӯжүҫгҖӮ

зҺ°еңЁи®©жҲ‘д»¬жқҘжөӢиҜ•дёӢеҗ§пјҡ

еңЁC:\DOMдёӯж–°е»әHelloWorld.java

йҰ–е…Ҳjavac HelloWorld.javaпјҢзі»з»ҹдјҡеңЁиҝҷдёӘзӣ®еҪ•дёӯжүҫеҲ°HelloWorld.javaж–Үд»¶пјҢзј–иҜ‘еҮәHelloWorld.classж–Үд»¶гҖӮJava HelloWorldпјҢзі»з»ҹдјҡе…Ҳи°ғз”ЁCLASSPATHи·Ҝеҫ„пјҢжҲ‘д»¬и®ҫзҪ®дәҶвҖң.вҖқиҝҷдёӘзӣ®еҪ•пјҢжүҖд»ҘдјҡеңЁеҪ“еүҚзӣ®еҪ•дёӯжүҫеҲ°HelloWorld.classгҖӮиҝҗиЎҢжҲҗеҠҹгҖӮ

**жҚўдёӘең°ж–№иҝҗиЎҢиҝҷдёӘзЁӢеәҸ

жҲ‘д»¬еңЁCзӣҳиҝҗиЎҢjava HelloWorld**

зі»з»ҹдјҡзҺ°еңЁCLASSPATHи·Ҝеҫ„еҜ»жүҫHelloWorld.classгҖӮдҪҶжҳҜеңЁвҖқ.вҖқпјҲзҺ°еңЁжҳҜCзӣҳпјүжүҫдёҚеҲ°иҝҷдёӘж–Үд»¶гҖӮ
жҲ‘д»¬зҺ°еңЁжҠҠC:\DOMи·Ҝеҫ„еҠ е…ҘеҲ°CLASSPATH

дёәдәҶжӣҙж–°зі»з»ҹзҺҜеўғеҸҳйҮҸжҲ‘д»¬йңҖиҰҒйҮҚж–°еҗҜеҠЁdosзӘ—еҸЈ

зі»з»ҹдјҡеңЁCLASSPATHдёӯжүҫеҲ°C:\DOMиҝҷдёӘи·Ҝеҫ„пјҢеҶҚжүҫеҲ°HelloWorld.classиҝҷдёӘж–Үд»¶гҖӮиҝҗиЎҢжҲҗеҠҹгҖӮ
зҺ°еңЁжҲ‘д»¬жҳҺзҷҪдәҶCLASSPATHзҡ„дҪңз”ЁдәҶ^^

е…ідәҺPACKAGE

жҲ‘д»¬жқҘж”№еҶҷиҝҷдёӘзЁӢеәҸ,еҠ е…Ҙpackage

зј–иҜ‘жҲҗеҠҹпјҢдҪҶжҳҜдёҚиғҪиҝҗиЎҢгҖӮ

зі»з»ҹеңЁжүҫеҲ°HelloWorld.classж—¶е°ұдјҡиҜ»еҸ–еҲ°package a.b;иҝҷдёӘдҝЎжҒҜпјҢиҝҷдёӘе‘ҠиҜүзі»з»ҹиҝҷдёӘHelloWorld.classжҳҜиҰҒж”ҫеңЁaж–Үд»¶еӨ№зҡ„bж–Үд»¶еӨ№дёӯгҖӮдҪҶжҳҜзҺ°еңЁжҲ‘д»¬зҡ„HelloWorld.classжҳҜеңЁDOMиҝҷдёӘж–Үд»¶еӨ№гҖӮж–Үд»¶и·Ҝеҫ„е’ҢpackageдёҚиғҪеҜ№еә”пјҢж— жі•иҝҗиЎҢгҖӮ

зҺ°еңЁжҲ‘д»¬жқҘж–°е»әa/bиҝҷдёӘдёӨдёӘж–Үд»¶еӨ№пјҢз„¶еҗҺжҠҠHelloWorld.classеӨҚеҲ¶еҲ°bж–Үд»¶еӨ№дёӯгҖӮз„¶еҗҺеҶҚиҝҗиЎҢjava HelloWorld

еҜ№дёҚиө·пјҒиҝҳжҳҜжІЎжңүиҝҗиЎҢжҲҗеҠҹгҖӮзі»з»ҹеңЁCLASSPATHдёӯеҜ»жүҫHelloWorld.classж–Үд»¶пјҢдҪҶжҳҜиҝҷдёӘж–Үд»¶иў«йҡҗи—ҸеңЁaж–Үд»¶еӨ№дёӯзҡ„bж–Үд»¶еӨ№гҖӮ
жҲ‘д»¬жқҘиҜ•иҜ•java a.b.HelloWorld

иҝҗиЎҢжҲҗеҠҹдәҶпјҢзі»з»ҹеңЁCLASSPATHдёӯзҡ„вҖң.вҖқж–Үд»¶еӨ№жүҫеҲ°aж–Үд»¶еӨ№пјҢз„¶еҗҺbж–Үд»¶еӨ№пјҢжүҫеҲ°HelloWorld.classж–Үд»¶гҖӮ
зҺ°еңЁжҳҺзҷҪpackageзҡ„дҪңз”ЁдәҶпјҢе®ғжҳҜдёҖе®ҡиҰҒз¬ҰеҗҲд»ҺCLASSPATHеҮәеҸ‘зҡ„зӣ®еҪ•з»“жһ„зҡ„гҖӮ

е…ідәҺIMPORT

Importзҡ„дҪңз”ЁжҳҜеҪ“дҪ и°ғз”ЁеҸҰдёҖдёӘclassзҡ„ж—¶еҖҷпјҢе‘ҠиҜүзі»з»ҹпјҢиҰҒжүҫе“ӘйҮҢжүҚиғҪжүҫеҲ°иҝҷдёӘclassгҖӮ

жҲ‘д»¬еңЁC:\DOMзҡ„зӣ®еҪ•дёӢеҶҷдёҖдёӘи°ғз”Ёa\b\HelloWorld.classзҡ„зЁӢеәҸгҖӮ

зј–иҜ‘иҝҗиЎҢжҲҗеҠҹпјҒ

зј–иҜ‘пјҡзі»з»ҹеңЁдёӘиҝҷдёӘж–Үд»¶еӨ№дёӯжүҫеҲ°TestHelloWorld.javaж–Үд»¶пјҢжЈҖжҹҘimport a.b.HelloWorld; жүҫCLASSPATHдёӯжүҫеҲ°a\b\HelloWorld.classгҖӮзј–иҜ‘йҖҡиҝҮе®ҢжҲҗпјҒ

иҝҗиЎҢпјҡзі»з»ҹзҺ°еңЁCLASSPATHдёӯжүҫеҲ°TestHelloWorld.classпјҢз„¶еҗҺиҜ»еҸ–import a.b.HelloWorld; еҜје…ҘHelloWorld.classпјҢиҝҗиЎҢжҲҗеҠҹпјҒ

еҲ°жӯӨдёәжӯўпјҢдҪ еә”иҜҘжҳҺзҷҪpackage е’Ң importзҡ„дҪңз”ЁдәҶгҖӮ
